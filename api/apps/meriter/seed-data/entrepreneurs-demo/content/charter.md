# Устав операционного фонда

Коллеги, добро пожаловать в **Сообщество предпринимателей**.

Мы договорились о простых правилах:

1. **Операционный фонд** — общий резерв для экстренной поддержки резидентов и совместных инициатив. Пополнение добровольное, решения о распределении — через голосование.
2. **Прозрачность** — каждую неделю казначей публикует отчёт: поступления, выплаты, остаток.
3. **Протокол решения** — сначала обсуждение в ленте, затем опрос с взвешенным голосованием, затем исполнение казначеем в течение 3 рабочих дней.
4. **Проекты** — крупные инициативы оформляем как кооперативные проекты с понятной экономикой.

Если согласны с логикой — поддержите пост заслугами, чтобы закрепить устав в ленте.
