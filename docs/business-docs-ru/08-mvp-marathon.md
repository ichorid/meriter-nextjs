# 08. MVP «Марафон Добра»

> Специфика текущего MVP: сообщества, настройки, флоу пользователя.

---

## О MVP

**Марафон Добра** — платформа для проведения игры, которую организовывают партнёры проекта.

**Целевая аудитория:** школьники

**Цель игры:** собрать как можно больше меритов и использовать их для поднятия вверх образов будущего (постов в ОБ), которые пользователю нравятся.

---

## Внутренняя лексика MVP

| Универсальный термин | В MVP называется |
|---------------------|------------------|
| Локальное сообщество | Группа команды |
| Лид локального сообщества | Лид команды |
| Глобальное сообщество | Публичная группа |

---

## Глобальные сообщества MVP

В MVP существует 4 глобальных сообщества:

### 1. Образ Будущего (ОБ)
**Главное сообщество игры.** Участники публикуют свои «образы будущего» и соревнуются за их продвижение.

**Ключевые настройки:**
| Настройка | Значение | Причина |
|-----------|----------|---------|
| Снятие меритов с постов | **Нет** | Мериты навсегда на посте |
| Голосование за свои посты | **Да** | Можно поддерживать свой ОБ |
| Квота | Нет | Тапалка — основной источник |
| Тапалка | Нет | Только голосование |
| Fee за пост | 1 мерит | Стандарт |
| Fee за коммент | 0.1 мерита | Стандарт |

**Особенность:** Периодически администрация поощряет авторов наиболее рейтинговых постов.

### 2. Марафон Добра (МД)
**Для отчётов о реально сделанных добрых делах.** Основной источник заработка меритов через голосования.

**Ключевые настройки:**
| Настройка | Значение | Причина |
|-----------|----------|---------|
| Снятие меритов с постов | **Да** | Основной способ заработка |
| Голосование за свои посты | Нет | Стандарт |
| Квота | **Нет** | Тапалка всё решает |
| Тапалка | **Да** | Основная эмиссия |
| Рейты | Планируется х2-х5 | Поощрение добрых дел |
| Fee за пост | 1 мерит | Стандарт |
| Fee за коммент | 0.1 мерита | Стандарт |

**Планируемые рейты:** При появлении дополнительных публичных групп (например, для развлекательного контента) — множитель для МД будет х2-х5, для развлекательного контента х0.1.

### 3. Проекты
**Для публикации идей и проектов «в процессе реализации».**

**Ключевые настройки:**
| Настройка | Значение | Причина |
|-----------|----------|---------|
| Снятие меритов с постов | **Нет** | Только после переноса в МД |
| Голосование за свои посты | **Да** | Можно поддерживать свой проект |
| Квота | Нет | — |
| Тапалка | **Да** | Эмиссия |
| Fee за пост | 1 мерит | Стандарт |
| Fee за коммент | 0.1 мерита | Стандарт |

**Особенность:** Когда проект выполнен, пост можно **перенести** в сообщество МД — и там с него снимутся все заслуженные мериты. Это единственный случай переноса с меритами.

### 4. Обратная связь
**Для предложений, вопросов и пожеланий к администрации.**

**Ключевые настройки:**
| Настройка | Значение | Причина |
|-----------|----------|---------|
| Fee за посты | **0** | Бесплатная обратная связь |
| Fee за комменты | **0** | Бесплатная обратная связь |
| Квота | Нет | — |
| Голосование пользователями | **Нет** | Только суперадмин начисляет |
| Снятие меритов | **Да** | Можно снять на кошелёк |
| Тапалка | Нет | — |

**Особенность:** Мериты начисляются **только суперадмином** — пользователи не могут голосовать за посты. При этом снимать мериты можно как обычно — так посты участвуют в экономике.

**Примечание:** Если начнут «спамить» — fee может быть включён.

## Основной флоу пользователя

### 1. Регистрация
1. Выбор способа авторизации
2. Заполнение профиля (имя, о себе)
3. Опционально: ввод инвайт-кода
4. Автоматическое получение:
   - Роль «Участник» во всех глобальных сообществах
   - 100 приветственных меритов
   - Доп. роль по инвайт-коду (если был)

### 2. Публикация идей
- Пользователь публикует свои образы будущего в **ОБ**
- Цель: собрать максимум меритов для своего образа

### 3. Заработок меритов

**Способ А: Добрые дела (МД)**
1. Сделать доброе дело в реальной жизни
2. Выложить отчёт в сообщество **МД**
3. Получить положительные комментарии от других
4. Снять заработанные мериты на кошелёк
5. Использовать для поддержки постов в ОБ

**Способ Б: Проекты**
1. Выложить идею/проект в сообщество **Проекты**
2. Собирать мериты через комментарии (нельзя снять)
3. Завершить проект в реальной жизни
4. Перенести пост в **МД**
5. Снять все заслуженные мериты

**Способ В: Тапалка**
1. Участвовать в тапалке (МД или Проекты)
2. За 10 сравнений получить 1 мерит
3. Использовать для поддержки постов

### 4. Трата меритов
- Оплата fee за публикации и комментарии
- Положительные комментарии к постам в ОБ
- Участие в опросах

### 5. Команды (опционально)
- Создать локальное сообщество (группу команды)
- Пригласить единомышленников
- Общаться, организовывать активности

### 6. Обратная связь
- Вопросы и предложения — в сообщество **Обратная связь**
- Суперадмин может начислить мериты за ценную обратную связь

## Роли в MVP

### Базовый путь
```
Регистрация
    │
    ▼
Участник всех глобальных сообществ
    │
    ├──► Вступить в локальное ──► Член команды
    │
    └──► Создать локальное ──► Лид команды
```

### Административные роли
- **Лид глобального сообщества** — выдаётся организаторам
- **Суперадмин** — техническая команда и главные организаторы

---

## Локальные сообщества в MVP

### Группы команд
- Создаются пользователями
- Все настройки на усмотрение лида
- Своя внутренняя валюта (отдельные от глобальных мериты)

### Ограничения
- Пересылка постов — всегда копия без меритов
- Лид не может удалить сообщество — только заявка
