# Сводка бизнес-решений: Глобальный мерит

> Дата: 13 февраля 2026
> Контекст: Унификация меритов в приоритетных сообществах и для fee

---

## Принятые решения

### 1. Глобальный мерит — определение

**Решение:** Один мерит на платформе для:
- Оплаты всех fee (postFee, commentFee, poll fee) во всех сообществах
- Голосования, вывода, тапалки, инвестиций в приоритетных сообществах

### 2. Приоритетные сообщества

**Определение:** Сообщества с общим глобальным меритом:
- Марафон Добра (marathon-of-good)
- Образ Будущего (future-vision)
- Проекты команд (team-projects)
- Поддержка (support)

**Расширяемость:** Новые приоритетные сообщества (через isPriority или typeTag) автоматически используют глобальный мерит.

### 3. Fee — источник

**Решение:** Fee во всех сообществах (включая локальные) списывается из глобального кошелька пользователя.

- В локальных сообществах: fee из глобального, голосование/участие — из локального кошелька
- Пользователь без глобальных меритов не может оплатить fee (даже в локальном сообществе)

### 4. Архитектура хранения

**Решение:** Синтетическое сообщество "Global" с id = `__global__`:
- Один кошелёк на пользователя: (userId, GLOBAL_COMMUNITY_ID)
- Модель Wallet без изменений — communityId может быть GLOBAL_COMMUNITY_ID
- Global community не показывается в списках сообществ, не требует членства

### 5. Миграция

**Подход:** Консолидация балансов:
- Для каждого пользователя: sum(МД, ОБ, Projects, Support) → глобальный кошелёк
- Старые кошельки приоритетных сообществ — обнулить или оставить для аудита (решается в реализации)
- Скрипт миграции с --dry-run, выполняется вручную до деплоя

### 6. Удаление sync МД↔ОБ

**Решение:** Полностью удалить syncDebitForMarathonAndFutureVision и syncCreditForMarathonAndFutureVision. Один глобальный кошелёк исключает необходимость синхронизации.

### 7. Локальные сообщества

**Без изменений:**
- Свой кошелёк (userId, communityId)
- Голосование, вывод — из локального кошелька
- Fee — из глобального (единственное изменение)

### 8. Квота (MVP)

В MVP квота отключена в приоритетных сообществах. Глобальная квота для приоритетных сообществ — отдельная задача при включении квоты.
