## Формат отчётов

Каждый отчёт `reports/0N-*.md` — по шаблону:

```markdown
# Этап N: [Название]

## Статус
✅ Завершён / ⚠️ Частично / ❌ Заблокирован

## Что сделано
- (конкретные пункты из тасклиста)

## Решения, принятые по ходу
- ⚠️ (описание и причина)

## Не удалось / заблокировано
- ❌ (если есть)

## Файлы созданные/изменённые
- `path/to/file` — что сделано

## Чеклист для проверки (человеком)
- [ ] (критические сценарии для ручной проверки)
```

Для **этапа 1 (backend модель)** обязательно добавить блок:
```
## CHECK-результаты
- CHECK-1: [что нашёл про merit-resolver]
- CHECK-2: [что нашёл про MembersList]
- CHECK-3: [что нашёл про эмиссию]
- CHECK-4: [что нашёл про профиль]
- CHECK-5: [что нашёл про навигацию сообщества]
```
